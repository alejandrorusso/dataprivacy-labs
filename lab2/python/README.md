# lab2/python

This directory contains the Python project template for Lab 2.

For a description of the lab assignment, see the Canvas page.

## Installing Python

The recommended way to install Python is through your OS's package manager.
We recommend using Python 3.11 or newer.

## Installing dependencies

This lab requires the Python packages `numpy` and `pytest`, which you can install either by running `pip install -r requirements.txt` in this directory, or through your OS's package manager. Depending on the package manager, they might be called `numpy` and `pytest` or `python3-numpy` and `python3-pytest`.
